### Show databases

```sql
SHOW DATABASES;
```

### Use specific database

```sql
USE database_name;
```

### Show information about columns from table

```sql
SHOW FIELDS from table_name;
```

### Show indexes from table

```sql
SHOW INDEXES FROM table_name;
```

### Insert row

```sql
INSERT INTO table_name (column_name_1, ...) VALUES (value_1, ...);
```
